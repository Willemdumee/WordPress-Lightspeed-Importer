== Description ==

Lightspeed Importer allows you to import products from Lightspeed .csv export file as simple as any other standard
WordPress import tool.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the directory `ctrl-lightspeed-importer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start to import your Lightspeed csv file
